# Drift

A terminal music player for Tidal, YouTube, and Bandcamp.

## Requirements

- mpv (audio playback)
- MPD (optional, for MPD integration)

## Usage

```sh
nix run .#drift
```

## Keybindings

| Key | Action |
|-----|--------|
| `Tab` | Switch panels |
| `j/k` | Navigate |
| `Enter` | Play |
| `Space` | Play/Pause |
| `n/N` | Next/Previous |
| `/` | Search |
| `q` | Quit |
